# Session Preview Console Panel

The static Operator Console includes a Local Session Preview panel for AION-095.
The panel uses `operator-console-static/demo-data/local-session-status.json` and
`operator-console-static/demo-data/local-session-preview.json`.

The panel displays:

- dev-only and read-only status
- production session disabled
- credential backing disabled
- token and cookie issuance disabled
- persistence disabled
- write, execution, activation, and external calls disabled
- synthetic expiry metadata

The static panel has no login form, logout form, password field, token field,
credential field, cookie field, local storage, session storage, or package
dependency.
