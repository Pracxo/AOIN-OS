# AION Brain v0.1 Operator Acceptance

AION means Adaptive Intelligence Orchestration Nexus. AION OS means Adaptive
Intelligence Orchestration Nexus Operating System.

## Acceptance Checklist

- [ ] Can start Docker stack.
- [ ] Can query health.
- [ ] Can query readiness.
- [ ] Can run setup doctor.
- [ ] Can run golden path.
- [ ] Can run RC gate.
- [ ] Can run demo script.
- [ ] Can inspect operator overview.
- [ ] Can validate generic extension manifest.
- [ ] Can confirm disabled boundaries.
- [ ] Can stop stack cleanly.

## Disabled Boundary Confirmation

The operator confirms that v0.1 does not enable production auth, full autonomy,
external model calls by default, external notification delivery by default,
extension code loading, capability activation, dynamic route registration, hard
delete, model-generated tool execution, or domain modules in Brain core.

## Required Operator Commands

```bash
docker compose up --build -d brain-api postgres redis nats opa
curl -fsS http://localhost:8080/health
curl -fsS http://localhost:8080/health/ready
./scripts/setup-doctor.sh --fast --offline-ok
./scripts/golden-path.sh --offline-ok
./scripts/rc-check.sh --offline-ok
./scripts/demo-local.sh --offline-ok
./examples/demo/operator-overview-curl.sh
docker compose down
```
