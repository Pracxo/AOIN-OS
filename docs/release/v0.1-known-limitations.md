# AION Brain v0.1 Known Limitations

AION means Adaptive Intelligence Orchestration Nexus. AION OS means Adaptive
Intelligence Orchestration Nexus Operating System.

- Local-first only.
- No production auth.
- No cloud deployment profile.
- No external notification delivery.
- No external model provider activation by default.
- No extension code loading.
- No capability activation.
- No domain modules.
- No UI operator console.
- No background scheduler loop.
- No automatic remediation.
- No hard delete.
- No legal/compliance retention guarantees.
- No performance SLA.

These limitations are release boundaries, not hidden functionality. AION Brain
v0.1.0 is a local baseline for future governed work.
