# v0.2 Implementation Request Evidence Checklist

## Required Checklist

- problem statement present
- risk statement present
- security impact present
- architecture impact present
- policy impact present
- audit/provenance impact present
- rollback plan present
- ADR dependency present
- gate dependency present
- test evidence present
- no-go acknowledgement present
- approval status false

## Checklist Decision

A complete checklist allows review to begin. It does not approve implementation, does not approve queue items, does not approve proposals, does not enable runtime, and does not create a v0.2 tag or release.
