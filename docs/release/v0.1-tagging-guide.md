# AION Brain v0.1 Tagging Guide

AION means Adaptive Intelligence Orchestration Nexus. AION OS means Adaptive
Intelligence Orchestration Nexus Operating System.

## Pre-Tag Checklist

- [ ] Required gates passed.
- [ ] Final docs audit passed.
- [ ] Freeze dry-run passed.
- [ ] Release package dry-run passed.
- [ ] Docker smoke passed when Docker is available.
- [ ] Operator acceptance passed.
- [ ] No no-go condition is present.

## Commands To Inspect Worktree

```bash
git status --short
git log -1 --oneline
git diff --check
```

## Commands To Run Final Gates

```bash
./scripts/v0.1-final-verify.sh
./scripts/v0.1-freeze.sh
```

## Commands To Create Local Tag

```bash
git status --short
./scripts/v0.1-final-verify.sh
git add -A
git commit -m "AION Brain v0.1.0 release baseline"
git tag aion-v0.1.0
git show --stat aion-v0.1.0
```

## Commands To Verify Tag

```bash
git show --stat aion-v0.1.0
git tag --list "aion-v0.1.0"
```

## Rollback Guidance If A Gate Fails

Do not tag if any required gate fails. Fix the failure, rerun the final gates,
and update evidence before commit. Do not push tag until operator sign-off. Do
not amend evidence after tagging without a new tag.
