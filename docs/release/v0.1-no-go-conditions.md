# AION v0.1 No-Go Conditions

Release is blocked if any of these conditions exist:

- Any failing required test
- Any failing SDK test
- Ruff failure
- mypy failure
- no-domain-drift failure
- boundary check failure
- policy coverage failure
- OpenAPI hygiene failure
- repo health failure
- Docker compose config failure
- health/readiness failure
- setup doctor critical finding
- golden path critical failure
- RC gate blocker
- freeze gate blocker
- release package dry-run failure
- final docs audit failure
- final verify script failure
- final freeze script failure
- tag preview mismatch
- external model calls enabled by default
- external notifications enabled by default
- extension code loading enabled
- capability activation enabled
- dynamic route registration enabled
- full autonomy enabled
- hard delete enabled
- production auth claimed as ready when incomplete
- raw secrets in docs or examples
- domain-specific module logic in Brain core
- any external publish, deploy, credential creation, or tag push before
  operator sign-off
