# ADR 0168: Controlled Knowledge Intelligence Research Acquisition Authorization

## Status

Accepted.

## Context

AION-203 completed the AION Cognitive Architecture Program with decision `CONTROLLED_LOCAL_OFFLINE_PILOT_PASS_COMPLETE_COGNITIVE_ARCHITECTURE_PROGRAM`. The completed architecture does not provide verified current public internet research or unrestricted world knowledge.

## Decision

Create `AION-KNOWLEDGE-INTELLIGENCE-001` and authorize AION-205 through `AION-204-KI-0001` to implement a disabled, allowlisted, operator-invoked research-acquisition and source-provenance core. AION-204 creates no research runtime source and makes no internet request.

## Consequences

Runtime research, background crawling, unrestricted network access, search-provider integration, connector integration, credential use, browser automation, automatic knowledge promotion, cognitive belief mutation, source mutation, Git mutation, runtime-created pull requests, automatic merge, production deployment, and model-weight training remain disabled. AION-206 must evaluate and close `AION-204-KI-0001` before any source-provenance registry authorization can exist.
