# Source Registry Operator Runbook

Operator use is read-only and local.

Allowed operator actions:

- Review static synthetic evidence.
- Run AION-207 focused tests.
- Run source-registry no-go, check, and runtime-hold gates.
- Replay an explicit synthetic fixture from an absolute local path outside the repository.
- Inspect deterministic indexes, integrity reports, incidents, diagnostics, and operator-review items.

Disallowed operator actions:

- Submit source content or URLs to the registry.
- Enable source-registry runtime.
- Persist registry data.
- Create a database or JSONL registry file.
- Verify factual claims.
- Promote knowledge.
- Mutate cognitive beliefs.
- Fetch from the network.
- Mutate source or Git.
- Create, approve, merge, or deploy through the registry.

Formal closeout and next authorization decisions belong to AION-208.
