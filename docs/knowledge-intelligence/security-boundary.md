# Knowledge Intelligence Security Boundary

AION-205 may implement only disabled, operator-invoked, allowlisted research acquisition after AION-204 merges. AION-204 performs no research network call.

Required AION-205 rules include URL parsing through a single canonical validator, only HTTPS by default, HTTP only for explicit synthetic local fixtures, explicit domain allowlist, lowercase IDNA-normalized hostnames, explicit port policy, no username or password in URLs, no URL fragments sent to the server, no local file paths, no environment-variable expansion, no shell command construction, no implicit proxy inheritance, no private address space, no loopback, no link-local, no multicast, no reserved address space, no cloud metadata endpoints, no IPv4-mapped IPv6 bypass, no decimal/octal/hexadecimal/mixed-form IP bypass, DNS resolution before request, destination validation after resolution, redirect destination revalidation, connection destination consistency check, bounded DNS results, fail closed on ambiguous resolution, TLS certificate validation, bounded response streaming, bounded decompression, compression-bomb protection, no automatic credential forwarding, no cookies, no authorization headers, no referer leakage, no cross-domain authentication, no HTML script execution, no browser storage, no service worker, and no arbitrary downloads.

Allowed methods: GET and HEAD only. POST, PUT, PATCH, DELETE, form submission, file upload, JavaScript execution, browser automation, login, cookies, credentials, content promotion, belief mutation, and background crawler execution are prohibited.

## AION-205 Controlled Research Acquisition Core

AION-205 implements the controlled research acquisition and immutable source-snapshot core as operator-invoked and runtime-disabled. Acquired content remains untrusted evidence; factual claim verification, knowledge promotion, cognitive belief mutation, public network fetch, crawler execution, search-provider integration, connector integration, source mutation, Git mutation, automatic merge, deployment, and model-weight training remain disabled. AION-204-KI-0001 remains active pending AION-206 closeout.
