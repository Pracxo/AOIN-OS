# Knowledge Intelligence Research Data Governance

All AION-204 examples are `synthetic=true`, `read_only=true`, and `redacted=true`.

AION-204 stores no credentials, tokens, cookies, authorization headers, private keys, raw prompts, hidden reasoning, raw user messages, personal data, live URLs, live domains, downloaded source content, source patches, or raw diffs.

AION-205 may create bounded source snapshots after AION-204 merges, but source content is never committed to the repository. Synthetic examples use reserved documentation domains only.
