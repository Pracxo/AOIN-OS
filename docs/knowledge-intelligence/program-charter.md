# AION Knowledge Intelligence Program Charter

Program ID: `AION-KNOWLEDGE-INTELLIGENCE-001`
Parent program ID: `AION-COGNITIVE-ARCHITECTURE-001`

Program objective: Build a governed epistemic intelligence fabric that allows AION OS to investigate permitted questions, acquire current evidence, verify factual claims, track provenance and contradiction, route work to domain specialists, use deterministic verification tools, and produce approval-bound verified knowledge candidates.

The program must separate:

1. Research acquisition
2. Source provenance
3. Claim extraction
4. Claim verification
5. Temporal and jurisdictional reasoning
6. Contradiction handling
7. Domain specialist reasoning
8. Deterministic tool verification
9. Verified knowledge memory
10. Engagement-learning candidates
11. Operator review
12. Runtime activation

## Core Epistemic Rule

Information is evidence.

Evidence may support a claim.

A claim becomes a verified knowledge candidate only after provenance, corroboration, freshness, contradiction, and policy checks pass.

A verified knowledge candidate remains reviewable and versioned.

No web page, user statement, engagement signal, model output, or repeated claim is automatically factual truth.

## Role Separation

Cognitive Architecture Program provides persistent state, a predictive world model, global workspace coordination, memory consolidation, counterfactual planning, active information-acquisition planning, governed continual-learning candidates, an integrated local-offline cognitive shadow runtime, and a passed controlled local-offline pilot. It does not provide unrestricted current world knowledge or verified public internet research and remains production-disabled.

The Cognitive Architecture Program does not provide verified public internet research.

Knowledge Intelligence Program acquires permitted external evidence, preserves source provenance, extracts atomic claims, binds claims to evidence, tracks time, jurisdiction, version, and contradictions, routes specialist work, uses deterministic verification tools, promotes only verified knowledge candidates, learns research methods from outcomes, does not treat repetition as truth, does not treat user statements as verified facts, and does not promote unverified web content into cognitive memory.
