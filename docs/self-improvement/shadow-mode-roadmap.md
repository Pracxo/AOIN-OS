# Shadow-Mode Roadmap

AION-177 authorizes only the next implementation task:

- AION-178: implement disabled, controlled, observation-only shadow-mode
  contracts, pipeline, evidence, budget, redaction, and operator review records.

Out of scope until separate authorization:

- Runtime source rewrite.
- Runtime self-improvement activation.
- Git mutation.
- Pull request creation.
- Automatic merge.
- Production canary or deployment.
- Provider or connector calls.
- Model-weight training.
- v0.2 tag or release.

Exit criteria for AION-178 must include consumption or closeout of
`AION-177-SI-0006`, runtime-disabled checks, no-go regression checks, and
operator-visible evidence proving shadow mode remains advisory only.
