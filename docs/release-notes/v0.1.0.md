# AION Brain v0.1.0 Release Notes

AION Brain v0.1.0 is the first local release freeze baseline for the generic
Brain core.

## Included

- Brain contracts, health, events, memory, policy, reasoning, planning,
  execution, visual telemetry, replay, regression, SDK, CLI, and release
  baseline surfaces.
- Version manifest and generic feature registry.
- Compatibility matrix and SDK compatibility report.
- Migration baseline and release artifact manifest.
- Deterministic freeze gate with local checks.
- Local release package, checksums, SBOM placeholder, and handoff report.

## Not Included

- Domain modules.
- External observability calls.
- Required optional adapters.
- Full autonomy by default.
- Frontend visualization implementation.

## Verification

Use the local check scripts and freeze gate:

```bash
./scripts/check.sh
./scripts/aionctl.sh --scope workspace:main freeze run --version 0.1.0
./scripts/aionctl.sh --scope workspace:main release package --version 0.1.0 --dry-run
./scripts/verify-release-package.sh
```
